📌 IIITL Mess Website
(Full-Stack Web Application | Group Project)
🧾 Overview

The IIITL Mess Website is a full-stack web application developed as a group project to digitally manage and display college mess services.
This platform provides an accessible, responsive, and user-friendly interface for students to view mess-related information efficiently.
This repository contains my implementation and contributions to the project.

🚀 Key Features

📱 Fully Responsive Design (Mobile, Tablet, Desktop)

♿ Implementation of 5 WCAG-based Accessibility Features

🔗 Integration of React Frontend with Spring Boot REST APIs

🎨 Consistent UI following provided style guide (colors, typography, layout)

⚡ Dynamic Data Rendering using API communication

🔄 Version Control using Git for collaborative development

▶️ How to Run the Project Locally
🔹 Backend Setup
cd backend
mvn spring-boot:run
🔹 Frontend Setup
cd frontend
npm install
npm start

Frontend runs on:
http://localhost:3000

Backend runs on:
http://localhost:8080
